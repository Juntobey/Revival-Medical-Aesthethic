## Installation

1. **Install Packages**  
   Run the following command to install the necessary packages:
   ```bash
   npm install


2. **Intialize local Database**  
   Run the following command to install the necessary packages:
   ```bash
  npx sequelize-cli db:migrate


2. **Start the App**  
   Run the following command to install the necessary packages:
   ```bash
  npm run start
